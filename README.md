Simple tool to test different AI models (GPT, Claude, Gemini) through the AI
Fiesta API.

## What it does

-   Test multiple AI models in one place
-   Real-time streaming responses
-   Saves your settings
-   Clean, simple interface

## Quick Start

1. Clone and install:

```bash
git clone <repository-url>
cd AI-Fiesta-Free
npm install
```

2. Start it:

```bash
npm run dev
```

3. Get your API token from [AI Fiesta](https://chat.aifiesta.ai) (check Network
   tab in DevTools)

4. Open http://localhost:3000 and start chatting!

## Available Models

-   GPT-4, GPT-3.5 Turbo
-   Claude 3.5 Sonnet
-   Gemini Pro
-   DeepSeek, Llama 3.1

## How to Use

1. Paste your Bearer token
2. Pick a model
3. Type your question
4. Hit send!
